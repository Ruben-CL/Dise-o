Descripción de la clase:

La clase está formada por 6 filas de mesas, con 3 ordenadores en cada una de ellas.
Además, entre oredenadores hay espacio para que se pueda trabajar también con portátiles, y las mesas cuentan con enchufes para facilitarlo.
En el lateral derecho hay tres ventanas exteriores, lo que favorece la buena ventilación y luminosidad de la clase.
Los profesores cuentan con un ordenador, dos monitores, una pizarra y un proyector para impartir sus asignaturas con garantía.
Por último, la clase dispone de gel y toallitas desinfectantes para poder cuidar la higiene y evitar contagios.

Módulos:

  1. Desarrollo web en entorno cliente
  2. Desarrollo web en entorno servidor
  3. Despliegue de aplicaciones web
  4. Diseño de interfaces web
  5. Empresa e iniciativa emprendedora
  6. Inglés técnico
